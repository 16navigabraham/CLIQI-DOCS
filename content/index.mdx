## What is CLIQI?

CLIQI is an autonomous trading agent that operates across X (Twitter) and Telegram. Two core features:

- **X agent** — Like a tweet containing a token address, CLIQI detects it and buys instantly. Your timeline becomes your trading terminal.
- **Telegram agent** — Add the bot to an alpha group, it watches for contract addresses from callers and auto-buys on your behalf. You can also buy, sell, and manage your full portfolio from DMs with the bot.

The platform runs on Base and Solana with embedded wallets — no seed phrases, no browser extensions.

## How It Works

**Via X (Like-to-Buy):**
```
User likes tweet on X → CLIQI detects like → Extracts contract address →
Validates token safety → Executes swap → User owns the token
```

**Via Telegram (DM):**
```
User sends contract address → CLIQI validates token → Executes swap →
Sends confirmation → Manage positions via /portfolio
```

**Via Telegram (Alpha Group):**
```
Alpha caller drops CA in group → CLIQI detects it → Auto-buys for
all group members with the agent enabled → Sends DM confirmations
```
