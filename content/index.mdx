## What is CLIQI?

CLIQI is a social-signal trading platform. Users connect their X (Twitter) account, and when they like a tweet containing a token contract address, CLIQI automatically executes a buy trade on their behalf. The platform supports Base and Solana chains with embedded wallets — no seed phrases, no browser extensions.

## How It Works

```
User likes tweet on X → CLIQI detects like → Extracts contract address →
Validates token safety → Executes swap → User owns the token
```

## Key Features

- **Like-to-buy automation** — Connect X, like a tweet, own the token
- **Multi-chain** — Base + Solana with chain auto-detection
- **Embedded wallets** — Powered by Privy (no seed phrases needed)
- **Multi-aggregator routing** — Best price across 0x, Odos, KyberSwap, Jupiter
- **USDC gasless trading** — Trade on Base without holding ETH
- **Auto-sell** — Take-profit & stop-loss monitoring (premium)
- **Telegram notifications** — Real-time trade alerts (premium)
- **PnL tracking** — Portfolio with realized/unrealized P&L and shareable trade cards
- **Narrative token launch** — AI detects trending narratives on X, launches tokens via Bankr, 70/30 fee buyback
- **Admin dashboard** — Real-time platform metrics
