## What is CLIQI?

CLIQI is a social-signal trading platform. Users sign up via X (Twitter) or Telegram, and when they like a tweet containing a token contract address, CLIQI automatically executes a buy trade on their behalf. The platform supports Base and Solana chains with embedded wallets — no seed phrases, no browser extensions. Telegram-first users can trade via the TG bot and connect their X account later to enable like-to-buy automation.

## How It Works

```
User likes tweet on X → CLIQI detects like → Extracts contract address →
Validates token safety → Executes swap → User owns the token
```

## Key Features

- **Like-to-buy automation** — Connect X, like a tweet, own the token
- **Telegram-first signup** — Create an account via TG, trade via bot, connect X later
- **Multi-chain** — Base + Solana with chain auto-detection
- **Embedded wallets** — Powered by Privy (no seed phrases needed)
- **Multi-aggregator routing** — Best price across 0x, Odos, KyberSwap, Jupiter
- **USDC gasless trading** — Trade on Base without holding ETH
- **Auto-sell** — Take-profit & stop-loss monitoring (premium)
- **Telegram notifications** — Real-time trade alerts (premium)
- **PnL tracking** — Portfolio with realized/unrealized P&L and shareable trade cards
- **Narrative token launch** — AI spots emerging trends across all of X (politics, AI, sports, world events), tokenizes them on Base via Bankr, 50/40/10 fee buyback
- **Alpha group copy-trading** — Add bot to private TG groups, track alpha callers, auto-buy when they drop CAs
- **Admin dashboard** — Real-time platform metrics
