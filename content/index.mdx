## What is CLIQI?

CLIQI is an autonomous trading agent that operates across X (Twitter) and Telegram. It has two core features: **like-to-buy on X** — like a tweet containing a token address and CLIQI automatically buys it, and **Telegram trading** — buy, sell, and manage your full portfolio directly from the TG bot. The platform supports Base and Solana chains with embedded wallets — no seed phrases, no browser extensions.

## How It Works

**Via X (Like-to-Buy):**
```
User likes tweet on X → CLIQI detects like → Extracts contract address →
Validates token safety → Executes swap → User owns the token
```

**Via Telegram:**
```
User sends /buy <token> in TG bot → CLIQI validates token →
Executes swap → Sends confirmation → User manages position via /portfolio
```
