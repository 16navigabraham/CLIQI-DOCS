# CLIQI

> **Like It. Buy It.** — The autonomous trading agent for X and Telegram. Executes trades from X likes and Telegram alpha group signals on Base and Solana.

**Version:** 1.0.0
**Chains:** Base (primary), Solana
**Website:** [cliqi.bot](https://cliqi.bot)

## Overview

CLIQI is an autonomous trading agent with two signal sources:

- **X agent** — Polls authenticated users' X likes via OAuth 2.0. When a liked tweet contains a contract address, the system validates token safety (GoPlus + RugCheck) and executes a swap through multi-aggregator routing.
- **Telegram agent** — Grammy bot monitors group messages from tracked alpha callers. On CA detection, executes buys for all subscribed users. DM interface exposes portfolio management, sell execution (inline buttons), and P&L tracking.

Runs on Base and Solana. Wallets are Privy server-side embedded wallets (no seed phrase export).

## Trade Flows

**X (Like-to-Buy):**
```
Like detected (polling) → CA extracted (ca-parser) → Safety check →
Price quote (multi-aggregator) → Swap execution → TX confirmation → Notify
```

**Telegram (Alpha Group):**
```
Group message from tracked caller → CA parsed → Dedup (Redis) →
Per-user buy via trade orchestrator → DM confirmation per user
```

## Core Systems

- **X like poller** — Per-user OAuth 2.0 polling (5s premium / 8s free), CA extraction from tweet text
- **Alpha group handler** — Grammy `message:text` listener, caller tracking per user per group (max 10), Redis message dedup
- **Trade orchestrator** — Validates → quotes → executes → confirms. Handles both buy and sell flows
- **Multi-aggregator routing** — 0x, Odos, KyberSwap (Base); Jupiter, PumpPortal (Solana). Races quotes, picks best
- **USDC gasless trading** — Treasury-sponsored gas on Base USDC sells
- **Auto-sell** — Background job (5s, 24/7) monitors premium users' positions against TP/SL thresholds
- **Telegram notifications** — Trade confirmations, P&L milestone alerts, narrative launches (premium)
- **PnL tracking** — Realised P&L from confirmed buys/sells, unrealised from live price feeds
- **Narrative engine** — Trend detection via X API → Grok scoring → Bankr token launch on Base → 100% ECHO buyback from creator fees
- **Admin dashboard** — Platform metrics, per-user stats, user search, time-range filtering
