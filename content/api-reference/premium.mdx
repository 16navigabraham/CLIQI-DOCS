# Premium API

## POST /api/premium/subscribe

Subscribe to premium using CLIQI token from your wallet.

**Auth:** Required

**Cost:** $10/month in CLIQI token

**Request Body:**
```json
{
  "paymentChain": "BASE"
}
```

Valid chains: `BASE`, `SOLANA`

**Success Response (200):**
```json
{
  "success": true,
  "message": "Premium subscription activated successfully!",
  "premiumUntil": "2026-04-19T12:00:00.000Z",
  "payment": {
    "chain": "BASE",
    "amount": 250000,
    "amountUsd": 10.00,
    "txHash": "0x5abc...",
    "explorerUrl": "https://basescan.org/tx/0x5abc..."
  }
}
```

**Errors:**
- `400` — Invalid chain, already subscribed, or insufficient CLIQI balance
- `404` — Wallet not found for specified chain
- `500` — Payment processing failed

## GET /api/premium/status

Get current premium subscription status.

**Auth:** Required

**Response:**
```json
{
  "isPremium": true,
  "premiumUntil": "2026-04-19T12:00:00.000Z",
  "daysRemaining": 28
}
```

## GET /api/premium/history

Get payment history for premium subscriptions. Returns last 20 payments.

**Auth:** Required

**Response:**
