# Wallet API

## GET /api/wallet

Get all wallets for the authenticated user.

**Auth:** Required

**Response:**
```json
{
  "wallets": [
    {
      "id": "clx...",
      "userId": "clx...",
      "chain": "SOLANA",
      "address": "7B5NrSkUQP...",
      "walletType": "EOA",
      "isPrimary": true,
      "isActive": true,
      "createdAt": "2026-01-01T00:00:00.000Z"
    }
  ]
}
```

## POST /api/wallet/create

Manually create a wallet (use if auto-creation failed during signup).

**Auth:** Required

**Request Body:**
```json
{ "chain": "SOLANA" }
```

Valid chains: `SOLANA`, `BASE`

**Response:**
```json
{
  "wallet": {
    "id": "clx...",
    "chain": "SOLANA",
    "address": "7B5NrSkUQP...",
    "isActive": true
  }
}
```

## POST /api/wallet/withdraw

Withdraw native tokens (SOL or ETH) to an external wallet address.

**Auth:** Required (+ passkey verification)

**Request Body:**
```json
{
  "chain": "SOLANA",
  "toAddress": "ExTernALwAlLetAdDrEss...",
  "amount": 0.1
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| chain | string | Yes | `SOLANA` or `BASE` |
| toAddress | string | Yes | Destination wallet address |
| amount | number | Yes | Amount in native units (SOL or ETH) |

**Success Response (200):**
```json
{
  "success": true,
  "txHash": "5xYzAbCd...",
  "chain": "SOLANA",
  "amount": 0.1,
  "toAddress": "ExTernALwAlLetAdDrEss..."
}
```

**Error Response (400):**
```json
{ "error": "Insufficient balance. Have 0.05 SOL, need 0.1 SOL + fee" }
```

**Explorer Links:**
- Solana: `https://solscan.io/tx/{txHash}`
- Base: `https://basescan.org/tx/{txHash}`

## GET /api/wallet/tokens

Get all token holdings with live prices. Zero-balance tokens are automatically excluded.

**Auth:** Required

**Response:**
```json
{
  "tokens": [
    {
      "chain": "SOLANA",
      "tokenAddress": "DezXAZ9HGe...",
      "tokenSymbol": "BONK",
      "tokenName": "Bonk",
      "amount": "1500000.5",
      "amountRaw": "15000005000000",
      "decimals": 5,
      "currentPriceUsd": 0.0000125,
      "currentValueUsd": 18.75
    }
  ]
}
```

For real-time updates, subscribe to the `prices:update` WebSocket event instead of polling this endpoint.
