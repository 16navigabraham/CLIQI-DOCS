# Wallet API

## GET /api/wallet

Get all wallets for the authenticated user.

**Auth:** Required

**Response:**
```json
{
  "wallets": [
    {
      "id": "clx...",
      "userId": "clx...",
      "chain": "SOLANA",
      "address": "7B5NrSkUQP...",
      "walletType": "EOA",
      "isPrimary": true,
      "isActive": true,
      "createdAt": "2026-01-01T00:00:00.000Z"
    }
  ]
}
```

## POST /api/wallet/create

Manually create a wallet (use if auto-creation failed during signup).

**Auth:** Required

**Request Body:**
```json
{ "chain": "SOLANA" }
```

Valid chains: `SOLANA`, `BASE`

**Response:**
```json
{
  "wallet": {
    "id": "clx...",
    "chain": "SOLANA",
    "address": "7B5NrSkUQP...",
    "isActive": true
  }
}
```

## POST /api/wallet/withdraw

Withdraw native tokens (SOL or ETH) to an external wallet address.

**Auth:** Required

**Request Body:**
```json
{
  "chain": "SOLANA",
  "toAddress": "ExTernALwAlLetAdDrEss...",
  "amount": 0.1
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| chain | string | Yes | `SOLANA` or `BASE` |
| toAddress | string | Yes | Destination wallet address |
| amount | number | Yes | Amount in native units (SOL or ETH) |

**Success Response (200):**
```json
{
  "success": true,
  "txHash": "5xYzAbCd...",
  "chain": "SOLANA",
  "amount": 0.1,
  "toAddress": "ExTernALwAlLetAdDrEss..."
}
```

**Error Response (400):**
```json
{ "error": "Insufficient balance. Have 0.05 SOL, need 0.1 SOL + fee" }
```

**Explorer Links:**
- Solana: `https://solscan.io/tx/{txHash}`
- Base: `https://basescan.org/tx/{txHash}`

## GET /api/wallet/tokens

Returns default wallet tokens (USDC, CLIQI, ECHO) with on-chain balances and live prices. Tokens are **always returned** even with zero balance — they act as the user's core wallet view. Traded tokens (from like-to-buy or manual swaps) appear in the portfolio instead (`GET /api/trades/portfolio`).

**Auth:** Required

**Default tokens:**
- **Base:** USDC, CLIQI, ECHO
- **Solana:** USDC

**Response:**
```json
{
  "tokens": [
    {
      "chain": "BASE",
      "tokenAddress": "0x833589fCD6eDb6E08f4c7C32D4f71b54bdA02913",
      "tokenSymbol": "USDC",
      "tokenName": "USD Coin",
      "amount": "50.00",
      "amountRaw": "50000000",
      "decimals": 6,
      "currentPriceUsd": 1,
      "currentValueUsd": 50.00,
      "isStable": true
    },
    {
      "chain": "BASE",
      "tokenAddress": "0x075b111329bd8fb54c6e2c1aa6d742a435ad7ba3",
      "tokenSymbol": "CLIQI",
      "tokenName": "Cliqi",
      "amount": "0",
      "amountRaw": "0",
      "decimals": 18,
      "currentPriceUsd": 0.00000268,
      "currentValueUsd": 0
    }
  ]
}
```

For real-time price updates, subscribe to the `prices:update` WebSocket event instead of polling this endpoint.
