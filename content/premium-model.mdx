## Payment Flow

Payment can be initiated from the **dashboard** (`POST /api/premium/pay`) or **Telegram** (`/premium` command).

### Base (CLIQI Token)

1. Fetch current CLIQI token price via DEXScreener
2. Calculate required amount: `$10 / CLIQI price`
3. Transfer CLIQI tokens from user's Base wallet to treasury
4. On-chain confirmation received
5. `isPremium = true`, `premiumUntil = now + 30 days`
6. `PremiumPayment` record created

### Solana (SOL)

1. Fetch current SOL price via price oracle
2. Calculate required amount: `$10 / SOL price`
3. Transfer SOL from user's Solana wallet to treasury
4. On-chain confirmation received
5. `isPremium = true`, `premiumUntil = now + 30 days`
6. `PremiumPayment` record created

## Premium Expiration

A background job checks every hour for expired subscriptions:

1. Finds users where `isPremium = true` AND `premiumUntil < now()`
2. Sets `isPremium = false`
3. Sends Telegram notification: "Your premium subscription has expired"
4. Auto-sell monitoring stops automatically
5. Polling speed reverts to 8s
6. Rate limits revert to free tier
7. Platform fee reverts to 1%
